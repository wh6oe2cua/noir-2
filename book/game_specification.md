# プロンプト・ノワール 仕様書
### 〜嘘つきAIの嘘を見抜け〜

## 1. ゲーム概要
本ゲームは、生成AI（嘘つきAI）やインターネットに溢れる「もっともらしい嘘」「巧妙な詐欺」「論理の飛躍」を見破る能力を養う、教育型のインタラクティブ・クイズゲーム（リテラシー・シミュレーター）です。
一見正しいように見える文章の中から、誤った一節をピンポイントで特定することで、プレイヤーの批判的思考力（クリティカルシンキング）とメディアリテラシーを格付けします。

---

## 2. コア・ゲームメカニクス
- **問題ジャンル選択**: プレイヤーは全6ジャンル（およびランダム）から挑戦したいテーマを選択可能。
- **文節タップシステム**: 提示される文章は自然な文節ごとにパーツ化（ボタン化）されており、プレイヤーはその中の「嘘・罠・飛躍」が含まれる1箇所をタップして判定します。
- **時間制限（40秒）**: 1問あたりの回答制限時間は40秒。タイムアップ時はライフが1減少します。
- **ライフシステム（3ハート）**: 誤判定または時間切れでライフ（ハート）が1つ減少します。ライフが0になるとゲームはリタイアとなります。
- **6ステージ構成**: 1ゲームは全6問で構成され、最終的な正解数（スコア）に応じてリテラシー格付け（称号）が決定されます。
- **重複防止エンジン**: 1回のゲームプレイ（6問）の中で、同じ問題が重複して出題されることがないよう、回答済み問題IDを追跡して除外する仕組みを搭載。

## 3. ジャンル構成とコンセプト
各ジャンルには、現代社会で騙されやすい実用的なトピックや、学術的・科学的な定番の誤解が用意されています。

| ジャンルID | ジャンル名 | 作問アプローチとコンセプト |
| :--- | :--- | :--- |
| `fraud_prevention` | **詐欺対策** | 還付金詐欺やフィッシングメール、SNS投資詐欺など、焦りや欲望を利用して人を騙す不審な言動や犯人の決定的な嘘を見抜く。 |
| `logic_flaw` | **論理の飛躍** | 「相関関係と因果関係の混同」「生存者バイアス」「前後即因果の誤謬」など、一見説得力があるが論理的に間違っている思考の罠を見抜く。 |
| `history` | **歴史** | 人名、日付、事件の主従関係などが巧妙にすり替えられた、歴史的な「もっともらしいデマ」を見抜く。 |
| `science` | **科学** | アマゾンの酸素供給率や脳の10%神話、宇宙ステーションの無重力の理由など、一般に広く信じられている科学的誤解を見抜く。 |
| `it_tech` | **IT・技術** | Bluetoothの接続制限、パスワードの定期変更方針、シークレットモードの保護範囲など、現代デジタル社会における仕様の勘違いを見抜く。 |
| `trivia` | **日常雑学** | ホッキョクグマの地肌、ヒマワリの開花後の向きなど、身近な生物や自然現象にまつわる有名な雑学の誤解を見抜く。 |

## 4. UI/UX & デザイン哲学
- **深夜の探偵事務所（フィルム・ノワール調）**:
  - 深みのあるチャコールやシャドウブラックを背景に、捜査資料を示す紙のようなクラフトベージュ、事件の手がかり（嘘）をハイライトする琥珀色（アンバー）や真鍮色を配色した、影と余白が織りなす大人の知的デザイン。
- **タイプライター・タイピング演出**:
  - 問題文が提示される際、タイプライターでカタカタと一文字ずつ文字が打ち込まれるようなレトロモダンなアニメーション（SE同調を含む）を搭載。
  - 単語の区切りごとに、古い書類にタイピングされたパーツのようなインダストリアルなチップUIとしてプレイヤーに提示。
- **捜査スタンプ調の直感的なフィードバック**:
  - プレイヤーが選択した箇所の判定時、ただの色変更だけでなく、捜査ファイルに「**RESOLVED（解決）**」や「**CONFIDENTIAL（極秘情報）**」「**FAILED（未解決）**」といった古いかすれスタンプが押されるようなビジュアル演出で、フィルム・ノワールの雰囲気を極大化。
- **詳細なファクトチェック解説**: なぜそれが嘘なのか、どのような事実が存在するのか、プレイヤーが日常生活で騙されないための「見破り方のコツ」を学習できるよう、1問ごとに丁寧な「捜査報告書（解説）」画面を提供。

## 5. 格付けシステム（リテラシー称号）
ゲーム終了時の最終スコアに応じて、プレイヤーに以下の格付け称号が付与されます：

1. **6点満点**: `AIマスターファクトチェッカー`
   - *説明*: 現代最高峰の情報強者。AIのハルシネーションや巧妙な罠を完全に見破った。
2. **5点**: `プロ級ファクトチェッカー`
   - *説明*: 極めて優秀なリテラシーの持ち主。冷静に物事を精査できる知性がある。
3. **3〜4点**: `中堅ファクトチェッカー`
   - *説明*: 平均以上の判断力。ただし巧妙に仕組まれた罠には時折騙される可能性がある。
4. **1〜2点**: `駆け出しファクトチェッカー`
   - *説明*: まだ甘い言葉や詐欺手口に惑わされがち。怪しいと思ったら調べる習慣が必要。
5. **0点**: `AIに騙されやすい一般市民`
   - *説明*: 危険信号。もっともらしい説明をすべて鵜呑みにしてしまうため、訓練が必要。
