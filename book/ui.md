# 3. 画面設計・UI

フィルム・ノワールの世界観（深夜の探偵事務所、事件の捜査資料）に基づき、高い没入感を持つレトロモダンなUIを設計します。

## 画面構成
アプリは以下の4つの基本画面（ステート）で構成されます。また、メインタイトルのすぐ横に小さく **「サウンド切り替え（音響オン/ミュート）」ボタン** を設置し、プレイヤーがいつでも音量をコントロールできるように配慮します。

1. **タイトル画面 (Start Screen) - 「深夜の探偵事務所」**
   - **背景演出**: 暗く静まり返ったチャコールグレー。すりガラス越しに街灯が差し込むようなグラデーション。
   - **ゲームタイトル**: 「プロンプト・ノワール」（クラシックなボールド体）
   - **サブタイトル**: 「〜嘘つきAIの嘘を見抜け〜」（モノスベースでレトロな副題）
   - **ルール説明 (捜査令状風)**:
     - 探偵事務所のデスクに広げられた黄ばんだ捜査令状（書類）を模したコンポーネント。
     - 制限時間60秒、ライフ3、全6問、ファクトチェックデータバンクから抽出された難事件であることを記載。
   - **事件ファイル（ジャンル）選択エリア**:
     - クラフト紙のインデックスフォルダー風ボタン。
     - 選択肢: `[歴史]` `[科学]` `[IT・技術]` `[日常雑学]` `[詐欺対策]` `[論理の飛躍]` `[ランダム]`。
   - **「捜査開始（ゲームスタート）」ボタン**: レトロモダンな真鍮色の点灯ボタン。

2. **ゲームプレイ画面 (Play Screen) - 「未解決事件の捜査ファイル」**
   - **ヘッダー**:
     - **残り時間インジケーター**: 白い煙が消えゆくかのような、あるいは古い時計を模したアナログ感覚のタイマー。
     - **ライフ表示**: 割れた電球や古いレコード盤、あるいはハートアイコン（ダメージを受けると粉々に砕け散るエフェクト）。
     - **進行状況**: 「CASE: 01 / 06」と表示され、1件ずつの捜査案件であることを強調。
   - **メインエリア**:
     - 「もっともらしく書かれた文章」が、**タイプライターでカタカタと一文字ずつ打ち込まれる**アニメーションと共に表示されます。
     - タイピング完了後、文章が文節ごとのインダストリアルなチップUI（古い印字テープ風、あるいは書類のファイリング用クリップ風）としてアクティブになります。
     - *例*: `[織田信長は]` `[1582年に]` `[本能寺の変において]` `[豊臣秀吉によって]` `[自害に追い込まれた。]`
   - **操作**:
     - プレイヤーが嘘（ハルシネーションの容疑箇所）だと思うパーツをタップし、犯人（犯節）を特定します。

3. **判定・解説画面 (Feedback Screen) - 「捜査結果報告」**
   - プレイヤーが特定した容疑箇所が「正解（嘘を見破った）」か「不正解（本当のことを嘘と疑った）」かを瞬時に判定します。
   - **正解（事件解決）の場合**:
     - 捜査資料に「**RESOLVED（解決）**」の緑色の捜査スタンプが力強く押されるエフェクト。
     - 選択したパーツが琥珀色（アンバー）から緑の蛍光インク風に染まります。
     - 正しい事実と解説文（ファクトチェック報告書）を表示。
   - **不正解（迷宮入り・誤認逮捕）の場合**:
     - 捜査資料に「**FAILED（未解決）**」または「**CONFIDENTIAL（極秘）**」の赤色のかすれたスタンプが押されるエフェクト。ライフが1減少。
     - 本来の間違いパーツが緑色で浮かび上がります。
     - 同様に、何が嘘で何が事実なのかを示すファクトチェック報告書を表示。
   - **「次の事件へ」ボタン**（タイプライター用紙がスクロールアップする演出）。

4. **リザルト画面 (Result Screen) - 「探偵の格付け・調書」**
   - **背景演出**: デスク上にずらりと並べられた事件解決ファイル。
   - **最終リポート（獲得スコア）**: 解決した事件数（例: 5 / 6 CASES SOLVED）を表示。
   - **探偵の格付け（リテラシー階級）**:
     - 捜査令状の上に、解決実績に応じた「ファクトチェッカー階級（例: AIマスターファクトチェッカー）」の公認スタンプや署名が刻印されます。
     - プレイヤーの分析能力やリテラシーを、レトロモダンな調書風の文体で評価します。
   - **「再捜査を行う（リトライ）」ボタン**。
